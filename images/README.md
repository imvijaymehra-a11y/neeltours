# Images Directory Structure

This directory contains local images for Neel Tour Travels website.

## Image Categories:

### Route Images:
- delhi-route.jpg - India Gate Delhi (Ambala to Delhi)
- manali-route.jpg - Manali Mountains (Chandigarh to Manali)  
- jaipur-route.jpg - Hawa Mahal Jaipur (Ambala to Jaipur)
- amritsar-route.jpg - Golden Temple Amritsar (Chandigarh to Amritsar)

### Logo:
- neel-tour-travels-logo.png - Company logo

### Service Images:
- local-taxi.jpg - Local taxi service
- airport-transfer.jpg - Airport transfer service
- outstation-taxi.jpg - Outstation taxi service
- corporate-travel.jpg - Corporate travel service

### Hero/Background:
- hero-bg.jpg - Hero section background

## Usage:
All images should be referenced as: `images/filename.jpg`
